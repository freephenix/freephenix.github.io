LeetCode-135. 分发糖果

题目

n 个孩子站成一排。给你一个整数数组 ratings 表示每个孩子的评分。

你需要按照以下要求，给这些孩子分发糖果：

每个孩子至少分配到 1 个糖果。
相邻两个孩子评分更高的孩子会获得更多的糖果。
请你给每个孩子分发糖果，计算并返回需要准备的 最少糖果数目 。

示例 1：

输入：ratings = [1,0,2]
输出：5
解释：你可以分别给第一个、第二个、第三个孩子分发 2、1、2 颗糖果。

示例 2：

输入：ratings = [1,2,2]
输出：4
解释：你可以分别给第一个、第二个、第三个孩子分发 1、2、1 颗糖果。
     第三个孩子只得到 1 颗糖果，这满足题面中的两个条件。

提示：

  • n == ratings.length
  • 1 <= n <= 2 * 10^4
  • 0 <= ratings[i] <= 2 * 10^4

思路

问题拆解

这个问题如果不做拆分可能会比较复杂，但是做一个拆分就会简单很多

  • 将每个位置需要关注两个方向的影响，拆解为分别从两个方向处理一遍数据，两边都符合的数据即是答案
  • 每个方向上的处理也均以题目中要求处理
    • 下一个评分高，那么就需要给下一个+1，当然如果本来就存在糖果数量差那就不需要再加了
    • 下一个评分低，那么就可以不操作，保留当前值
    • 所有位置初始化值均为 1

这样一来问题就会简单很多

Tips

第二趟走过时，可以顺便完成总数的求解，节约一趟遍历

上代码，细节在注释了～

代码

func candy(ratings []int) int {
	// 问题拆分，每个点都应该符合两个方向上的规则
	// 那就分别从两个方向处理一遍，使得数据符合规则即可

	// 初始化数据均为1
	candyNum := make([]int, len(ratings))
	for i := range candyNum {
		candyNum[i] = 1
	}

	// 先从左往右走一遍
	for i := 1; i < len(ratings); i++ {
		if ratings[i] > ratings[i-1] {
			candyNum[i] = candyNum[i-1] + 1
		}
	}

	// 从右往左走一遍，顺便可以统计总糖数
	count := candyNum[len(candyNum)-1]
	for i := len(ratings) - 2; i >= 0; i-- {
		if ratings[i] > ratings[i+1] && candyNum[i] <= candyNum[i+1] {
			candyNum[i] = candyNum[i+1] + 1
		}
		count += candyNum[i]
	}
	return count
}
