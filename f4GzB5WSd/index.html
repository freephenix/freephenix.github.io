LeetCode-69. x 的平方根

题目

给你一个非负整数 x ，计算并返回 x 的 算术平方根 。

由于返回类型是整数，结果只保留 整数部分 ，小数部分将被 舍去 。

注意：不允许使用任何内置指数函数和算符，例如 pow(x, 0.5) 或者 x ** 0.5 。

示例 1：

输入：x = 4
输出：2

示例 2：

输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。

提示：

  • 0 <= x <= 231 - 1

思路

平方根不用内置函数的话，正向思维去求解会比较麻烦，那我们反过来想，如果一个数a是x的平方根，那么a跟x有什么关系呢？
x = a*a，
而如果a仅代表平方根整数部分的话，将满足
a*a <= x 且 (a+1)*a+1 > x

确定了判定条件，那么就看如何找a了，
从0到x逐个遍历固然可行，但是二分查找将会更有效率。

至此，上代码。

代码

func mySqrt(x int) int {
	l, r := 0, x
	ans := 0
	for l <= r {
		mid := l + (r-l)/2
		if mid*mid <= x {
			ans = mid
			l = mid + 1
		} else {
			r = mid - 1
		}
	}
	return ans
}
